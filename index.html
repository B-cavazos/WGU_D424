Vacation Planner

Vacation Planner is a native Android application developed in Java as the capstone project for Western Governors University's Bachelor of Science in Software Engineering program.

The application allows users to organize vacations and excursions while demonstrating Android development concepts including Room persistence, RecyclerViews, CRUD operations, search functionality, reporting, notifications, and data validation.

Features

  • Create, edit, and delete vacations
  • Create, edit, and delete excursions
  • Search vacations and excursions
  • Generate vacation reports
  • Local Room database persistence
  • Vacation start and end notifications
  • Share vacation details using Android Share Sheet

Technologies

  • Java
  • Android SDK
  • Room Persistence Library
  • RecyclerView
  • Material Components
